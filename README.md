**游戏说明**  
C++实现的贪吃蛇功能，双击MySnake.exe开始游戏      
  
***注意：***  请务必将bgMusice.wma、win.wma、socredata.txt放在与EXE文件同一目录下  
  
**项目GitHUB地址：** https://github.com/StoveJunJun/Snake  
  
**说明**  
MySnake.cpp 主程序实现  
Game.cpp 游戏逻辑实现部分  
Logo.cpp 开场动画实现部分  
Map.cpp  游戏地图的实现部分  
Snake.cpp 蛇实现部分  
Food.cpp 食物实现部分  
Point.cpp  使用光标绘制  
Setdisplay.cpp 使用Writechar绘制
